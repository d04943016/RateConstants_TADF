📖 Theoretical Background

1. Three-Level Kinetic Model

A TADF emitter has three states: S₀ (ground), S₁ (singlet), T₁ (triplet). Excitons populate S₁ or T₁ upon excitation (fraction α to S₁). From S₁, emission (ksr), non-radiative decay (ksnr), or ISC (kISC) can occur. From T₁, thermally activated RISC (kRISC) or non-radiative decay (kTnr).

dS₁/dt = −ks·S₁ + kRISC·T₁ + αG
dT₁/dt = kISC·S₁ − kt·T₁ + (1−α)G
ks = ksr+ksnr+kISC   kt = kTnr+kRISC

The PL decay is biexponential: I(t) = BPF·e−t/τPF + BDF·e−t/τDF, arising from the two eigenvalues of the coupled system:

kPF = (ks+kt+Δ)/2   (fast, prompt fluorescence)
kDF = (ks+kt−Δ)/2   (slow, delayed fluorescence)
Δ = √[(ks−kt)² + 4·kISC·kRISC]

2. Rate Constants

SymbolPhysical meaning
ksrRadiative decay S₁→S₀; governs prompt fluorescence yield
ksnrNon-radiative S₁ decay (internal conversion, IC)
kISCIntersystem crossing S₁→T₁; driven by spin-orbit coupling
kTnrNon-radiative T₁ decay; main efficiency loss pathway
kRISCReverse ISC T₁→S₁; thermally activated — the TADF key step

3. Directly Determined from Experiment

ksr = kPF·ΦPF + kDF·ΦDF
ks = (kPF²·ΦPF + kDF²·ΦDF) / ksr
kt = (ΦPF+ΦDF)·kPF·kDF / ksr
kISC·kRISC = ΦPF·ΦDF·kPF·kDF·(kPF−kDF)² / ksr²

4. ΦTnr Sweep (Underdetermined Parameter)

kISC and kRISC individually require ΦTnrPL (non-radiative triplet loss fraction) as an additional constraint. The calculator sweeps ΦTnr from 0 to (1 − PLQY):

kISC = (kISC·kRISC + ΦTnr·kPF·kDF) / kt
kRISC = (kISC·kRISC) / kISC
ksnr = ks − ksr − kISC

5. Quantum Efficiencies

Φsr = ksr/ks   ΦISC = kISC/ks
ΦRISC = kRISC/kt   ΦTnr = kTnr/kt
PLQY = Φsr / (1 − ΦISC·ΦRISC)

6. Worked Example

Input: τPF = 15 ns, τDF = 2.9 µs, ΦPF = 70%, ΦDF = 12%
kPF = 6.67×10⁷ s⁻¹,   kDF = 3.45×10⁵ s⁻¹

Directly determined:
ksr ≈ 4.67×10⁷ s⁻¹,   ks ≈ 6.66×10⁷ s⁻¹,   kt ≈ 4.04×10⁵ s⁻¹

At ΦTnr = 0:   kISC ≈ 9.65×10⁶, kRISC ≈ 4.04×10⁵ s⁻¹, kTnr = 0
At ΦTnr = 18%: kRISC decreases, kTnr increases — set by PLQY constraint
