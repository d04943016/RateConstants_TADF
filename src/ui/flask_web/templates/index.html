TADF Rate Constant Calculator

Three-level excitonic system — singlet & triplet state dynamics

Theoretical Background

Three-level photophysical model for TADF emitters (Wu et al. 2015–2017)

1. Three-Level Kinetic Model

A TADF emitter is described by three states: ground state S0, lowest excited singlet S1, and lowest triplet T1. Upon photoexcitation, excitons in S1 can either emit prompt fluorescence (radiative decay ksr), undergo non-radiative decay (ksnr), or undergo intersystem crossing to T1 (kISC). From T1, excitons can return to S1 via thermally activated reverse ISC (kRISC), decay non-radiatively (kTnr), or emit phosphorescence (kTr ≈ 0 for efficient TADF).

dS1/dt = −ks · S1 + kRISC · T1 + αG
dT1/dt = kISC · S1 − kt · T1 + (1−α)G
where ks = ksr + ksnr + kISC , kt = kTr + kTnr + kRISC

The biexponential PL decay observed experimentally arises from the two eigenvalues of this coupled system:

I(t) = BPF · exp(−t/τPF) + BDF · exp(−t/τDF)

kPF = (ks + kt + Δ) / 2
kDF = (ks + kt − Δ) / 2
Δ = √[(ks − kt)² + 4·kISC·kRISC]

3. Directly Determined Rate Constants

From the measured (kPF, kDF, ΦPF, ΦDF), three quantities can be determined directly without any additional assumption:

ksr = kPF·ΦPF + kDF·ΦDF

ks = (kPF²·ΦPF + kDF²·ΦDF) / ksr

kt = (ΦPF + ΦDF) · kPF · kDF / ksr

kISC·kRISC = ΦPF·ΦDF·kPF·kDF·(kPF−kDF)² / ksr²

4. Underdetermined Rate Constants & the ΦTnr Parameter

kISC and kRISC individually cannot be determined from PL data alone because only their product is known. An additional parameter is needed: ΦTnrPL, the fraction of excitons lost non-radiatively from T1 under PL excitation (α = 1).

kISC(ΦTnr) = (kISC·kRISC + ΦTnr·kPF·kDF) / kt
kRISC(ΦTnr) = kISC·kRISC / kISC
ksnr(ΦTnr) = ks − ksr − kISC
kTnr(ΦTnr) = kt − kRISC

The calculator sweeps ΦTnrPL from 0 to its maximum physical value (= 1 − PLQY), tracing how all rate constants and efficiencies depend on this unknown parameter. The true operating point lies somewhere along this curve.

5. Efficiency Definitions

Φsr = ksr / ks     (radiative yield from S1)
Φsnr = ksnr / ks    (non-radiative loss from S1)
ΦISC = kISC / ks    (ISC efficiency)
ΦRISC = kRISC / kt   (RISC efficiency)
ΦTnr = kTnr / kt    (triplet non-radiative loss)

PLQY = Φsr / (1 − ΦISC·ΦRISC)

6. Worked Example

Using default parameters: τPF = 15 ns, τDF = 2.9 µs, ΦPF = 70%, ΦDF = 12%, PLQY = 82%

Inputs:
kPF = 1/15 ns = 6.67×107 s−1
kDF = 1/2.9 µs = 3.45×105 s−1
ΦPF = 0.70, ΦDF = 0.12

Directly determined:
ksr = 6.67×107×0.70 + 3.45×105×0.12 ≈ 4.67×107 s−1
ks ≈ 6.66×107 s−1 (≈ kPF since kPF ≫ kDF)
kt ≈ 4.04×105 s−1

At ΦTnr = 0 (best case):
kISC ≈ 9.65×106 s−1, kRISC ≈ 4.04×105 s−1
ksnr ≈ 1.02×107 s−1, kTnr ≈ 0

At ΦTnr = 18% (worst case, = 1−PLQY):
kISC and kRISC both decrease; kTnr increases
→ Efficiency ΦRISC drops significantly

7. Input Modes

Lifetime mode (τ)

Enter τPF in nanoseconds and τDF in microseconds. The calculator converts via k = 1/τ. Requires quantum yield inputs ΦPF, ΦDF as percentages (sum ≤ 100%).

Rate constant mode (k)

Enter kPF directly in units of 108 s−1 and kDF in 105 s−1.

B-prefactor mode

Use when fitting the decay curve as I(t) = BPF·exp(−t/τPF) + BDF·exp(−t/τDF). Enter BPF, BDF, and total PLQY; the calculator converts to ΦPF, ΦDF via weighted integration: Φi = PLQY × (Bi·τi) / (BPF·τPF + BDF·τDF).
