📖 Theoretical Background

1. Three-Level Kinetic Model

A TADF emitter has three states: S₀ (ground), S₁ (singlet), T₁ (triplet). From S₁: radiative emission (ksr), non-radiative decay (ksnr), or ISC (kISC). From T₁: thermally activated RISC (kRISC) or non-radiative decay (kTnr).

dS₁/dt = −ks·S₁ + kRISC·T₁ + αG
dT₁/dt = kISC·S₁ − kt·T₁ + (1−α)G
ks = ksr+ksnr+kISC   kt = kTnr+kRISC

PL decay: I(t) = BPF·e−t/τPF + BDF·e−t/τDF. The two observed decay times arise from eigenvalues of the coupled S₁/T₁ system:

kPF = 1/τPF = (ks+kt+Δ)/2
kDF = 1/τDF = (ks+kt−Δ)/2
Δ = √[(ks−kt)² + 4·kISC·kRISC]

2. Rate Constants

SymbolPhysical meaning
ksrRadiative decay S₁→S₀; governs prompt fluorescence yield
ksnrNon-radiative S₁ decay (internal conversion)
kISCIntersystem crossing S₁→T₁; driven by spin-orbit coupling
kTnrNon-radiative T₁ decay; main efficiency loss pathway
kRISCReverse ISC T₁→S₁; thermally activated — the TADF key step

3. Directly Determined Constants

Four quantities are uniquely determined from τPF, τDF, ΦPF, ΦDF alone:

ksr = kPF·ΦPF + kDF·ΦDF
ks = (kPF²·ΦPF + kDF²·ΦDF) / ksr
kt = (ΦPF+ΦDF)·kPF·kDF / ksr
kISC·kRISC = ΦPF·ΦDF·kPF·kDF·(kPF−kDF)² / ksr²

4. ΦTnr Sweep

kISC and kRISC individually require ΦTnrPL (non-radiative triplet loss fraction). The calculator sweeps ΦTnr from 0 to (1−PLQY):

kISC = (kISC·kRISC + ΦTnr·kPF·kDF) / kt
kRISC = (kISC·kRISC) / kISC
ksnr = ks − ksr − kISC

5. Quantum Efficiencies

Φsr = ksr/ks   ΦISC = kISC/ks
ΦRISC = kRISC/kt   ΦTnr = kTnr/kt
PLQY = Φsr / (1 − ΦISC·ΦRISC)

6. Worked Example

Input: τPF=15 ns, τDF=2.9 µs, ΦPF=70%, ΦDF=12%
kPF=6.67×10⁷ s⁻¹, kDF=3.45×10⁵ s⁻¹

Directly determined:
ksr≈4.67×10⁷, ks≈6.66×10⁷, kt≈4.04×10⁵ s⁻¹
kISC·kRISC≈3.91×10¹² s⁻²

At ΦTnr=0: kISC≈9.65×10⁶, kRISC≈4.04×10⁵ s⁻¹
At ΦTnr=18%: kRISC decreases, kTnr increases
