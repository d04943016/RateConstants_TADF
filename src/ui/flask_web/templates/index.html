Theoretical Background▾

1. Three-Level Kinetic Model

A TADF emitter has three states: S₀ (ground), S₁ (singlet), T₁ (triplet). Excitons in S₁ emit prompt fluorescence (ksr), undergo non-radiative decay (ksnr), or cross to T₁ (kISC). From T₁, excitons return via thermally activated RISC (kRISC) or decay non-radiatively (kTnr).

dS₁/dt = −ks·S₁ + kRISC·T₁ + αG
dT₁/dt = kISC·S₁ − kt·T₁ + (1−α)G
ks = ksr+ksnr+kISC   kt = kTnr+kRISC

The biexponential PL decay I(t) = BPF·e−t/τPF + BDF·e−t/τDF arises from eigenvalues:

kPF = (ks+kt+Δ)/2
kDF = (ks+kt−Δ)/2
Δ = √[(ks−kt)² + 4·kISC·kRISC]

2. Rate Constants

SymbolPhysical meaning
ksrRadiative S₁ decay; controls ΦPF
ksnrNon-radiative S₁ loss (internal conversion)
kISCS₁→T₁ intersystem crossing; spin-orbit coupling
kTnrNon-radiative T₁ loss; limits TADF efficiency
kRISCT₁→S₁ thermally activated upconversion (key TADF)

3. Directly Determined

ksr = kPF·ΦPF + kDF·ΦDF
ks = (kPF²·ΦPF+kDF²·ΦDF)/ksr
kt = (ΦPF+ΦDF)·kPF·kDF/ksr
kISC·kRISC = ΦPF·ΦDF·kPF·kDF·(kPF−kDF)²/ksr²

4. ΦTnr Parameter

kISC and kRISC individually require an additional assumption: ΦTnrPL (triplet non-radiative loss fraction). The calculator sweeps ΦTnr from 0 to 1−PLQY.

kISC = (kISC·kRISC + ΦTnr·kPF·kDF) / kt
kRISC = kISC·kRISC / kISC
ksnr = ks − ksr − kISC

5. Efficiencies

Φsr = ksr/ks   ΦISC = kISC/ks
ΦRISC = kRISC/kt   ΦTnr = kTnr/kt
PLQY = Φsr/(1−ΦISC·ΦRISC)

6. Worked Example

Input: τPF=15 ns, τDF=2.9 µs, ΦPF=70%, ΦDF=12%
kPF=6.67×10⁷ s⁻¹, kDF=3.45×10⁵ s⁻¹

Directly determined:
ksr≈4.67×10⁷ s⁻¹, ks≈6.66×10⁷ s⁻¹, kt≈4.04×10⁵ s⁻¹

At ΦTnr=0: kISC≈9.65×10⁶, kRISC≈4.04×10⁵ s⁻¹
At ΦTnr=18%: kRISC drops, kTnr rises
